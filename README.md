# 暑期实训：可信证书存证系统

本仓库用于完成暑期实训创新项目 1：基于区块链的实训证书与学业证明存证系统。

项目当前已收束为：

> 证书生命周期管理系统 + 哈希存证验真系统。

两周开发优先保证完整闭环：模板配置、Excel 导入、PDF 证书生成、二维码分享、SHA-256 哈希、本地哈希链或测试链存证、扫码验真、撤销补发和审计日志。

## 关键文档

- `AGENTS.md`：本项目 Codex 协作规则和开发边界。
- `docs/项目规划/开发目标收束.md`：当前开发目标、范围、风险、里程碑和验收标准。
- `docs/项目规划/目录说明.md`：当前目录结构和后续建议目录。
- `docs/原始材料/`：课程任务书原始副本。
- `docs/提交模板/`：课程要求提交的 Word 文档模板。

## 当前优先级

1. 先完成业务闭环，不先追求复杂区块链。
2. 优先使用 SHA-256 + 本地哈希链完成可信验真演示。
3. 确保证书编号、PDF、二维码、哈希、回执、撤销状态能互相追踪。
4. DID/VC、Merkle Root、真实链部署、多机构协同作为提高项或后续扩展。
