# 目录说明

当前项目目录用于保存暑期实训“可信证书存证系统”的资料、规划、模板和后续代码。

## 目录结构

```text
暑期实训/
├─ AGENTS.md                      # 本项目 Codex 协作规则
├─ docs/
│  ├─ 项目规划/                    # 收束后的项目目标、路线、决策记录
│  ├─ 原始材料/                    # 课程发布的原始选题说明副本
│  ├─ 提交模板/                    # 学校要求提交的 Word 模板
│  └─ 过程文档/                    # 提取文本、分析过程、中间材料
```

## 文件说明

- `docs/项目规划/开发目标收束.md`：当前最重要的项目范围文件，后续开发优先遵循。
- `AGENTS.md`：给 Codex 和团队成员看的项目协作规则。
- `docs/原始材料/`：保存课程任务书副本，避免后续整理时误改原始要求。
- `docs/提交模板/`：保存需求、概要设计、详细设计和测试计划模板。
- `docs/过程文档/`：保存从 Word 中提取出的文本和分析脚本。

## 后续建议目录

开始编码后建议新增：

```text
backend/       # 后端服务
frontend/      # 前端页面
database/      # 建表脚本、初始数据、ER 图
scripts/       # 导入、生成、部署、演示脚本
tests/         # 自动化测试或测试数据
outputs/       # 生成的演示证书、截图、答辩产物
```
